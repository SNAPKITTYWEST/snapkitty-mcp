SNAPKITTY SOVEREIGN OS - BUILD LANE - MCP

Sovereign AI tools
for Claude and Cursor.

5 MCP tools that give your AI assistant direct access to the SnapKitty sovereign stack: WORM sealing, agent builds, Ada contracts, twin chat, and context injection. Drop into Claude Desktop in 2 minutes.

View on GitHub npm package
QUICK INSTALL - 2 STEPS
1
Install the package
npm install -g snapkitty-mcp
2
Add to Claude Desktop config (claude_desktop_config.json)
{ "mcpServers": { "snapkitty": { "command": "snapkitty-mcp" } } }
3
Restart Claude Desktop. All 5 tools appear in the tool palette.

SOVEREIGN TOOLS

worm_sealSeal a payload onto the WORM chain. SHA-256 hash, Ed25519 signature, timestamp. Returns the block receipt.chain
worm_readRead the WORM chain tail. Returns the last N sealed blocks with hashes and signatures.chain
agent_buildScaffold a new sovereign agent with role profile, soul spec, and trust deed template.agents
ada_contract_generateGenerate an Ada 2022 contract spec for a given agent. Preconditions, postconditions, invariants.formal
twin_chatChat with a digital twin agent using the sovereign prompt envelope. EDAULC trust score applied.agents
sovereign_injectInject sovereign context into any agent session. Borrow chain, kernel state, trust registry.kernel
